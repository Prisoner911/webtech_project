Put attendance folder in : "C:/xampp/mysql/data".

Put webtech folder in : "C:/xampp/htdocs".

NOW FOLLOW THESE STEPS:

NOTE: IF YOU DON'T HAVE MYSQL WORKBENCH INSTALLED THAN SKIP TO STEP 4.

1) If you have mysql workbench installed then first press "windows key + R".
2) Now type services.msc in that box and click ok.
3) Now search for MySQL80 and right click on it then click on stop.
4) Now open XAMPP control panel, start apache and mysql in it.
5) Now open chrome browser and type "localhost:80/webtech/index.html"
